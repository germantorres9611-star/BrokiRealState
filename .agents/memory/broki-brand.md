# Broki brand palette

- Background: `#0D3140` → `hsl(200, 66%, 15%)` (deep teal/navy)
- Primary accent: `#8FA84D` → `hsl(77, 37%, 48%)` (olive sage green)
- Foreground: near-white `hsl(200, 8%, 94%)`
- Font: **Nunito** (rounded, friendly — matches the logo exactly)
- The full site is dark mode to match the brand identity

**Why:** User provided the official logo showing dark teal background + white rounded text + olive green accent. Previous palette (white + forest green) was wrong.

**How to apply:** Always use `hsl(200, 66%, 15%)` as `--background`, `hsl(77, 37%, 48%)` as `--primary`. Keep Nunito as both display and body font.
